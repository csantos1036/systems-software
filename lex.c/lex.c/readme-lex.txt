To compile and run the VM, the input file name will be read as a command line argument at runtime, such as $ ./a.out test1.txt. The program will be printed to the screen. Every lexeme and token will be printed line by line. At the end of the program, every token will be printed as a list.

STEPS: 
1) $ gcc lex.c
2) $ bash tester.sh 
or 
1) $ gcc lex.c
2) $ ./a.out test1.txt.
